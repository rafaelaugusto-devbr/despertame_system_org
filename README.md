Desperta-me
Bem-vindo ao repositório do site Desperta-me, uma organização sem fins lucrativos dedicada a promover valores cristãos e fortalecer a fé por meio de eventos, conteúdos e ações comunitárias.

Sobre o Projeto
O site Desperta-me foi desenvolvido para ser a principal plataforma de comunicação da organização, onde são divulgados eventos, notícias, projetos sociais e conteúdos formativos. Ele visa facilitar o acesso dos membros e interessados às atividades e informações relevantes, além de oferecer um painel administrativo para facilitar a gestão dos conteúdos.

Funcionalidades
Página inicial com informações institucionais.

Sessão de eventos com lista dinâmica e possibilidade de cadastro, edição e exclusão pelo painel administrativo.

Área administrativa protegida por login, para gerenciar eventos e conteúdos.

Integração com Firebase para autenticação, armazenamento de dados e imagens.

Layout responsivo e de fácil navegação.

Tecnologias Utilizadas
HTML5, CSS3 e JavaScript puro para a estrutura, estilo e funcionalidades do front-end.

Firebase (Authentication, Firestore, Storage) para backend e gerenciamento dos dados.

Hospedagem via Firebase Hosting.
